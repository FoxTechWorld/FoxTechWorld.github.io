Como Substituir O Hamachi No Linux Para Jogar Minecraft Lan De Forma Facilitada

November 11, 2023

Como substituir o hamachi no linux para jogar minecraft LAN de forma facilitada

O que é hamachi

Se você já quis jogar num mundo survival com amigos, provavelmente já usou o hamachi que é um aplicativo para Windows que serve principalmente para que simula uma rede local ou Local Area Network permitindo que pessoas se conectem de forma facilitada imitando LAN e é normalmente é utilizada para jogos no:

  • FreeBSD
  • Linux
  • MacOSX
  • Windows

Primeira solução: NGROK

Ngrok,a primeira vez pode parecer um pouco mais complicado por não ter interface gráfica mas ele é superior ao hamachi sendo um proxy reverso distribuído globalmente suportando entregas via HTTP / TLS ou aplicações baseadas em TCP.

Isso significa que você pode usar TCP/IP pra usar ngrok e compartilhar um link com seu amigo e ambos podem jogar minecraft juntos.

O uso é simples, após você instalar o ngrok em seu sistema operacional basta abrir a Lan no seu mapa minecraft que vai receber a porta contendo 5 dígitos. após isso é só usar o comando

ngrok tcp 127.0.0.1:{porta}

Temos opções extras como configurar o ngrok para usar apenas conexões de um determinado continente garantindo melhor performance do servidor

ngrok tcp 127.0.0.1:{porta} --region sa

Para mais informações visite a documentação oficial do ngrok~

Segunda solução: Localtunnel

Uma opção mais aberta ao Hamachi e ao Ngrok, se chama Localtunnel, diferente do ngrok temos menos configurações de acesso o que nos permite menor granularidade de configurações, em contrapartida para abrir o servidor passamos apenas a porta do servidor e não seu ip completo com

lt --port {porta}

inclusive, você pode pegar e estudar o código do Localtunnel através de seu github: Github do Localtunnel

Você não consegue especificar as regiões onde quer que o Localtunnel atue, pois seu funcionamento é diferente do Ngrok que é diferente do Hamachi, mas todos servem para fazer uma conexão entre amigos via tunnel.

Salve menção a Fabio Akita que explica o conceito de tunnel com ssh.

Conclusão

Com isso podemos concluir diversas opções que podem substituir o hamachi como seu software para jogar lan entre amigos com diversos outros que além de serem cli, podem ser expandidos e adaptados para rodar via script.

Qualquer dúvida que tiverem é só deixar nos comentários, compartilhe com seus amigos e flw.
