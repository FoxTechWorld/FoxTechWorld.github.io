Lista Ligada

A lista ligada é uma das estruturas mais simples na matéria de estruturas de dados.

E ouso dizer que também chega a ser uma das mais importantes.

Pois com ela podemos criar estruturas muito mais complexas.

Um exemplo simples que podemos dar a lista ligada em C++ é:

Class Node{
    private:
        int Value;
        Node* next;
    public:
        Node(int new_value){
            Value = new_value;
            next = NULL;
        }
};

Onde para aqueles que possuem um olho atento. Podem ver que a lista pode se expandir infinitamente para o seguinte. De forma linear, ou O(1) para cada elemento e O(N) para o tamanho máximo da lista.
